**Project Overview**
The Smart Monitoring System for House Construction Projects is a digital solution designed to tackle challenges in residential construction management. By integrating real-time progress tracking, automated reporting, financial oversight, and centralized resource coordination, the system addresses inefficiencies in manual processes and fragmented communication.

**Objectives**
The main objectives of developing this system are to:

- Identify and analyse key requirements for a smart dashboard specifically designed for house construction project monitoring.
- Design a smart dashboard that supports real-time tracking, interactive data visualization, and efficient user interactions.
- Develop a dashboard for monitoring house construction projects, integrating features for real-time updates, task tracking, and visual representation of progress data.
- Evaluate how the dashboard improves coordination, communication, and decision-making among project stakeholders.

**Tools & Technologies Used**

₊⊹Frontend: HTML, CSS, JavaScript
₊⊹Backend: PHP
₊⊹Database: PHPMyAdmin
₊⊹Data Visualization: Chart.js

**Types of Users Involved**

- Top Management
- Project Manager
- Site Supervisor
- House Owner

**Key Features**

1. Real-Time Monitoring
- Visualize construction progress, project milestones and delay risks.
- Enable project managers to track overall project status and make timely adjustments.
- Allow site supervisors to monitor and update daily tasks and respond quickly to issues.
- Provide clients with clear updates.

2. Data Support with Visual Insights
- Interactive visualizations display key metrics such as timeline progress, and budget management.
- Assist project managers in making data-driven decisions to keep projects within scope, budget, and schedule.
- Allow clients to receive real-time updates.
- Help supervisors and team members quickly identify and address potential issues, improving responsiveness.

3. Communication and Collaboration
- Built-in messaging or chat feature for seamless communication between project stakeholders.
- Facilitate coordination, reporting, and technical support among project teams.

4. User and System Administration
- Top Management can manage user accounts and define access levels for all users
